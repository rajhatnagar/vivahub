# VivaHub

## Getting Started

To access the project, navigate to:  
[http://localhost/vivahub/](http://localhost/vivahub/)

## Credentials

**Admin Access**  
Email: `admin@vivahub.com`  
Password: `password`

**User Access**  
Email: `user@vivahub.com`  
Password: `password`

## Setup

1. Copy `.env.example` to `.env`
2. Run `composer install`
3. Run `php artisan key:generate`
4. Run `php artisan migrate:fresh --seed`
